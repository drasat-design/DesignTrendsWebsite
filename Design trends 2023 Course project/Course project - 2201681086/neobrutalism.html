Neubrutalism

Neubrutalism takes recent website trends to the next level—it retains its edge while also combining the unusual style with more modern animation, illustration, and text standards. Although neubrutalism is an unofficial term right now, it’s already being talked about in the world of web design and it’s the sort of thing which sticks, so it's great to get ahead of the curve!


Characteristics:

  • Clashing colours

  • No gradients

  • Unique typography

  • Animations
